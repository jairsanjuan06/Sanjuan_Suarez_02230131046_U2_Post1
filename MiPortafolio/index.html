Visión General del Proyecto

Este portal documenta el ciclo de vida del desarrollo de una suite de cuatro videojuegos educativos orientados a la concienciación sobre enfermedades contagiosas. El objetivo es utilizar la gamificación para la retención de información crítica de salud pública.

Tecnologías y Módulos Desarrollados

Módulo 1: Simulación de Propagación (COVID-19)

Desarrollo enfocado en renderizado tridimensional para visualizar el comportamiento del virus en diferentes entornos cerrados.

  • Motor/Lenguaje: JAVA 3D
  • Estado: En fase de validación técnica.

Módulo 2: Vectores y Prevención (Zika, Dengue, Fiebre Amarilla)

Suite de minijuegos interactivos de rápida ejecución en navegadores y lógica de backend para gestión de puntajes.

  • Frontend: JavaScript (Lógica de interfaz y DOM)
  • Backend/Lógica: Python

Métricas de Arquitectura y Gestión

La viabilidad y ejecución de este sistema de software se apoya en los siguientes pilares de gestión:

  1. Duración del Sprint/Ciclo: Estimación de 12 meses de desarrollo continuo.
  2. Asignación de Recursos: Presupuesto proyectado de $550 millones para infraestructura, licencias y desarrollo.
  3. Validación de Datos: Auditoría estricta de la información médica implementada en los juegos.

Requerimientos Activos

Nota para reclutamiento: Actualmente el proyecto requiere la validación de un experto en salud pública para auditar la precisión clínica de los algoritmos de contagio dentro de los videojuegos.
